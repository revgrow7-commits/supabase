**Key Technical Concepts**
- **Modularização Completa:** Conclusão da migração de uma arquitetura monolítica para uma baseada em micro-roteadores (FastAPI APIRouter).
- **Correção de Bugs Full-Stack:** Habilidade de rastrear um bug da UI (ex: contagem 0) até sua origem na query do banco de dados no backend e corrigi-lo em todas as camadas.
- **Backfill de Dados:** Uso de scripts únicos para corrigir dados históricos no banco de dados após uma correção na lógica de negócio (ex: cálculo de tempo de check-in).
- **Desenvolvimento Orientado a Funcionalidades:** Implementação de funcionalidades complexas de ponta a ponta, como o arquivamento, que envolveu alterações no backend, na API e em múltiplos componentes de UI.
